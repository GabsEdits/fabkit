# Fabkit — Component Roadmap

## In Progress / Planned

### Input & Form

- [x] `TextArea` — multi-line, auto-resize, floating label
- [x] `NumberField` — stepper +/- arrows, min/max/step
- [x] `Checkbox` — native-feel with label, animated check
- [x] `RadioGroup` — single-selection group
- [x] `TagsInput` — inline add/remove tags
- [x] `ColorPicker` — hex input + swatch + hue slider
- [ ] `DatePicker` — calendar dropdown GNOME-style
- [x] `SearchField` — TextField with built-in clear + icon
- [x] `FileChooser` — drop zone + click, shows selected files

### Feedback & Status

- [ ] `Toast` — temporary notifications with optional action
- [ ] `ProgressBar` — linear, determinate and indeterminate
- [ ] `Spinner` — circular loading indicator
- [ ] `Badge` — dot/count overlaid on other elements
- [ ] `Banner` — inline contextual alert (info/warning/error/success)

### Navigation

- [ ] `Breadcrumb` — file-manager-style path navigation
- [ ] `Toolbar` — contextual action row above content
- [ ] `CommandPalette` — Ctrl+K overlay with fuzzy search
- [ ] `Pagination` — prev/next + page numbers

### Layout

- [x] `Separator` — horizontal/vertical divider with optional label
- [x] `Accordion` — collapsible expander (libadwaita ExpanderRow style)
- [x] `Tooltip` — hover label, auto-positioned

### Data Display

- [ ] `Avatar` — circle with image or initials fallback
- [x] `DataTable` — sortable table with selection and row actions
- [ ] `TreeView` — file tree / nested expandable list
- [x] `Chip` — removable, colorable pill
- [x] `Kbd` — styled keyboard shortcut display

### Native Desktop (libadwaita / GNOME HIG)

- [x] `ActionRow` — row with title + subtitle + right widget
- [x] `PreferencesGroup` — group of ActionRows with section title
- [x] `PreferencesPage` — full settings page with groups
- [x] `EntryRow` — ActionRow with inline TextField
- [x] `SwitchRow` — ActionRow with integrated Switcher
- [x] `SpinRow` — ActionRow with integrated NumberField
- [x] `LevelBar` — segmented bar (battery, storage, quota)
