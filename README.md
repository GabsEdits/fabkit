# Fabkit

Fabkit is a UI toolkit and component library.

Install:
- npm: `npm install github:fabricatorsltd/fabkit#main`
- pnpm: `pnpm add github:fabricatorsltd/fabkit#main`

Usage:
```js
import { Button } from 'fabkit';
```

Development:
- Install deps: `pnpm install`
- Build: `pnpm build`
- Test: `pnpm test`
